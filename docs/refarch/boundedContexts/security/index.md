# Security BC

The security BC will handle all request with regard to the following:

1. Authentication
2. Authorization
3. User management (menus, profiles, roles and permissions ) i.e. AIM
4. SSL termination.
5. PIN translation.
6. Application management.
7. Link incryption.
8. PKI (pem)
9. Basic authentication (client id, secret)
10. JWT
11. generique crypto
12. Audit
13. Logging
